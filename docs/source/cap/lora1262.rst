LoRa1262 Cap
============

.. sku: U214

.. include:: ../refs/cap.lora1262.ref

Cap LoRa1262 is a high-performance LoRa communication and GNSS global navigation expansion module designed for the Cardputer-Adv.

Support the following products:

    |LoRa1262Cap|
