**********
UI
**********

M5Paper具有触摸功能，您可以直接点击右侧的选项卡来切换到相应的应用程序。

|flow.png|

在CODE界面，您可以看到自己设备的UIFlow模式，MAC地址，账号信息。您还可以在UI左下角确认WiFi和服务器的连接情况。

|config.png|

在SETTING界面，您可以看到自己的WiFi配置信息和服务器地址。

|applist.png|

在APPLIST界面，您可以直接点击文件名，并点击右侧的运行按钮运行指定的代码。
