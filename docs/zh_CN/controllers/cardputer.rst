**********
Startup UI
**********

|launcher.png|

**左** 和 **右** 键循环浏览可用的应用程序。

**Enter** 启动选定的应用程序。

|setting.png|

**左** 和 **右** 键循环浏览可用的应用程序。

**Enter** 启动选定的应用程序。

**ESC** 和 **BtnA** 退出应用程序。

|develop.png|

**ESC** 和 **BtnA** 退出应用程序。

|apprun.png|

**A** 键运行应用程序一次。

**O** 键始终运行应用程序。

**ESC** 和 **BtnA** 退出应用程序。

|applist.png|

**向上** 和 **向下** 键在可用的应用程序之间循环。

**A** 键运行应用程序一次。

**O** 键始终运行应用程序。

**ESC** 和 **BtnA** 退出应用程序。
