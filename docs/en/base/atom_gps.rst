Atomic GPS Base
================

.. sku: A134/K043

.. include:: ../refs/base.gps.ref

This is the driver library of ATOM GPS Base, which is used to obtain data from the
GPS module.

Support the following products:

    ================== ==================
    |ATOM GPS|         |ATOM GPS Base|
    ================== ==================
